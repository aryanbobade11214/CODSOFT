# Task 2 - Sales Prediction (CODSOFT Internship)

## 📌 Project Description
This project is part of the CodSoft Data Science Internship.  
The goal is to predict sales based on advertising and other features using Machine Learning.

## 📊 Dataset
The dataset contains information about:
- Advertising spend (TV, Radio, Newspaper)
- Sales output

## ⚙️ Steps Performed
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Model Training (Linear Regression / ML model)
5. Prediction & Evaluation

## 🧠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

## 📁 Files in this folder
- dataset.csv (data file)
- model.ipynb or script.py (code file)

## 🚀 Outcome
Successfully built a model to predict sales based on input features.
